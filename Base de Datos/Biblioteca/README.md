# Biblioteca - Proyecto de Base de Datos

Este proyecto de base de datos ha sido desarrollado en SQL Server y tiene como objetivo gestionar una biblioteca. Incluye la creación de esquemas, tablas, vistas, consultas y triggers.

## Estructura del Proyecto

El código SQL se ha dividido en varios archivos para facilitar la navegación y comprensión. A continuación, se presenta la estructura del proyecto:

1. **01_Creacion_Base_Datos.sql:** Este archivo reinicia la base de datos y realiza la creación inicial.

2. **02_Creacion_Schemas_Tablas.sql:** Contiene los scripts para crear esquemas y tablas necesarios para la biblioteca.

3. **03_Vistas.sql:** Incluye la definición de vistas que proporcionan información útil sobre la biblioteca.

4. **04_Consultas.sql:** Contiene consultas SQL que permiten realizar búsquedas y obtener información específica de la base de datos.

5. **05_Triggers.sql:** Incluye triggers que automatizan ciertos procesos dentro de la base de datos.

## Instrucciones de Uso

1. Ejecuta el archivo `01_Creacion_Base_Datos.sql` para reiniciar la base de datos.
2. Luego, ejecuta los archivos `02_Creacion_Schemas_Tablas.sql`, `03_Vistas.sql`, `04_Consultas.sql` y `05_Triggers.sql` en ese orden para completar la configuración.
3. Utiliza el proyecto según tus necesidades.

## Notas Importantes

- Asegúrate de tener SQL Server instalado y configurado antes de ejecutar los scripts.
- Revisa y ajusta las configuraciones según tus necesidades específicas.
