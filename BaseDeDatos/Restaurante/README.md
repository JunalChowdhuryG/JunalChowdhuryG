# Base de Datos de Restaurante

Este proyecto consiste en una base de datos diseñada para un restaurante, implementada en MySQL/MariaDB. Proporciona una estructura básica para administrar pedidos, reservaciones, cuentas y más.


## Archivos SQL

El proyecto incluye varios archivos SQL para crear y poblar la base de datos y definir vistas:

1. **01_Creacion_Base_Datos.sql**: Este script reinicia la base de datos, eliminando y volviendo a crear todas las bases de datos necesarias.
2. **02_Creacion_Tablas.sql**: Define las tablas necesarias para la base de datos.
3. **03_Completar_Tablas_1.sql**, **04_Completar_Tablas_2.sql** : Estos scripts llenan las tablas con datos ficticios proporcionados por IA y algoritmos aleatorios en Python.
4. **05_Vistas.sql**: Define una serie de vistas que ofrecen diferentes perspectivas sobre los datos de la base de datos.
